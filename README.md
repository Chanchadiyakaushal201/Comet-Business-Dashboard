# Comet Business Dashboard (Power BI)

## Project Overview

This project is an end-to-end Power BI dashboard designed for a sneaker brand (**Comet**) to analyze business performance across **Sales, Operations, and Finance**.

The dashboard provides actionable insights to support decision-making by identifying growth trends, operational inefficiencies, and profitability drivers.

---

## Objectives

* Analyze overall business performance
* Monitor delivery efficiency and operational issues
* Evaluate profitability and return impact
* Support data-driven business decisions

---

## Dataset Description

The dataset includes:

* Orders data (sales, delivery status, delays)
* Product data (category, model, pricing)
* Customer data (membership, location)
* Calendar table for time-based analysis

---

## Key Metrics (KPIs)

* Total Sales
* Profit & Profit %
* Total Orders
* OTIF % (On-Time In-Full)
* Avg Delivery Days
* Avg Late Days
* AOV (Average Order Value)
* Return Rate %

---

## Dashboard Structure

### 1. Overview Page (Business Performance)

Focus: High-level summary

Key Insights:

* Sales & Profit growth (YoY)
* Monthly sales trends
* Top-performing categories and cities
* Delivery performance distribution

---

### 2. Operations Page (Delivery Analysis)

Focus: Root cause of operational issues

Key Insights:

* Delivery efficiency (On-time vs Late)
* Delay trends over time
* Top delay cities
* Category-wise delay hotspots
* Order outcome distribution

---

### 3. Finance Page (Profitability Analysis)

Focus: Financial health of business

Key Insights:

* Profit margin vs target
* Average Order Value (AOV)
* Return rate analysis
* Profit by category
* Top products by profit
* Revenue split (Member vs Non-member)

---

## Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
* Data Cleaning (Power Query)

---

## Key Insights

* Strong sales and profit growth overall
* High late delivery percentage impacting OTIF
* Return rate higher than target (needs attention)
* Certain categories drive both profit and delays
* Non-member customers contribute majority revenue

---

## Project Steps

1. Data Cleaning & Transformation (Power Query)
2. Data Modeling (Relationships, Calendar Table)
3. KPI Creation using DAX
4. Dashboard Design (UI/UX with brand theme)
5. Visual Selection & Optimization
6. Insight Generation

---

## Design Approach

* Minimal black & white theme (aligned with brand identity)
* Clean layout with intuitive navigation
* Focus on clarity and decision-making
